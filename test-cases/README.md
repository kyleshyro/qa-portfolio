# Test Cases

Manual test cases demonstrating structured QA thinking.

## Files
- `manual-test-cases.md` — Login, regression, and data validation test cases

## Format Used
Each test case includes: Test Case ID, Feature, Type, Priority, Preconditions, Steps, Expected Result, and Notes.

## Test Types Covered
- ✅ Functional (Happy Path)
- ✅ Negative Testing
- ✅ Boundary Value Testing
- ✅ Security Testing (SQL Injection)
- ✅ Regression Testing
- ✅ Data Validation
